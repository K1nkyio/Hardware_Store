# Raph Supply User Dashboard

Customer-facing storefront for Raph Supply. Built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

## Local development

```sh
npm install
npm run dev
```

The dev server runs on `http://localhost:8081` by default.

## Build

```sh
npm run build
```

## Main areas

- Home merchandising and category discovery
- Product detail and add-to-cart flow
- Checkout and payment handoff
- Customer authentication and account management
- About, customer care, privacy, and terms pages

## Notes

- API requests are configured through `VITE_API_BASE_URL`
- Static storefront assets live in `public/`
- Shared UI components live under `src/components/ui`
